              MixedList

Multiple Type storage List

This kind of list can store every XMLObject type declared in its parent ChoiceNode. Plus Strings.

Example:

class Foo(XMLObject):
    pass

class Bar(XMLObject):
    mix = ChoiceNode(['#PCDATA','Foo'], noLimit=True)

In mix node, a backend MixedList will be created. It will ensure items stored are either strings ('#PCDATA') or Foo instances.
